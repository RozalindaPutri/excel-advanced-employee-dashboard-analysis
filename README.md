# 📊 excel-advanced-employee-dashboard-analysis

## 🚀 Project Overview
This project showcases an **advanced employee data analysis workflow** using Microsoft Excel.  
The analysis focuses on processing employee data, creating helper pivot tables, building pivot charts, designing an interactive dashboard, and using VBA Macro to automate data cleaning.

The case study demonstrates how raw employee data can be cleaned, summarized, analyzed, and visualized into a clear dashboard to support better decision-making related to employee status, KPI performance, salary contribution, bonus calculation, gender distribution, and employee hiring trends.

## 🔑 Key Features
* 🧹 Data Cleaning using Excel Macro
* 🧾 Helper Pivot Table for employee analysis
* 📊 Pivot Table summary for key business questions
* 📈 Pivot Chart for employee visualization
* 👥 Gender-based employee percentage analysis
* 📅 Historical employee hiring trend analysis
* 💰 Salary contribution analysis by position
* 🎁 Bonus calculation based on KPI rules
* 🖥️ Employee Dashboard Reporting
* 💻 VBA Macro with Message Box notification

## 📊 Analysis Focus
* 📌 Identifying permanent employees based in Jakarta
* 📌 Analyzing employees approaching retirement age
* 📌 Categorizing employee KPI performance
* 📌 Calculating salary contribution by job position
* 📌 Calculating total bonus by job position based on KPI requirements
* 📌 Visualizing employee percentage by gender
* 📌 Visualizing historical employee hiring trends by year
* 📌 Cleaning blank rows automatically using VBA Macro
* 📌 Presenting all key insights in a dashboard
